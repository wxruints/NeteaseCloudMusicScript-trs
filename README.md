## 自定义部分功能并开关某些不需要的歌词版本的简便方法

**·**  调整优先级：修改 **lrc_order** 数组中 **不同格式** 的位置即可（如**移到最后则优先级最低**）
​
**·**  修改翻译括号：直接改 **bracket** 数组，例如 [" **{** ", " **}** "] 则输出 [00:08.24] 原词 **{** 翻译 **}** 
​
**·**  删除该版本：从 **lrc_order** 数组中 **直接删除** 想要删除的格式，**不影响** 其他格式
​
**·**  调整搜索数量：修改 **limit** 数值，**数字越大搜索结果越多**，匹配 **成功率越高**
